Made using [Astro](https://astro.build/). 
    
Features:
----
 - [Vue](https://vuejs.org/) Components using [@astrojs/vue](https://docs.astro.build/en/guides/integrations-guide/vue) for the UI.
 - [MDX](https://mdxjs.com/) Support using [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx) for content.
 - [Nano Stores](https://github.com/nanostores/nanostores#nano-stores) with [@nanostores/vue](https://github.com/nanostores/vue) for state management.

#### Project Structure

```
.
├── src # Source files
│   ├── components # Vue Components
│   ├── layouts # Astro Layouts
│   ├── pages # Astro Pages
│   ├── styles # CSS
│   └── utils # Utilities
├── public # Public files
├── .gitignore # Git Ignore
├── astro.config.mjs # Astro Config
├── package.json # Package.json
├── README.md # Readme
└── tsconfig.js # TypeScript Config
```

Portfolio Creation
Recently I've been working on my portfolio (this website) to share my projects and experiences.
Published: Jul 8, 2022
Last updated on Mar 31, 2023

I’ve started this portfolio and blog to share my projects and experiences with the world, built as a static site using Astro and Vue making it fast and easy to maintain while allowing me to choose any front-end framework I want.

Currently the site is hosted on GitHub Pages, meaning it’s open-source and accessible to anyone who wants to view it. Using git also allows it to integrate seamlessly with Astro’s content system, making it easy to deploy and update my site as I add content. You can find the source code for this site on GitHub .
